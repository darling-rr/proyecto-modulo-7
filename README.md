# Proyecto Módulo 6 - Node & Express Web App

## Descripción
Este proyecto corresponde a la primera parte del desarrollo de una aplicación backend con Node.js y Express.  
En esta etapa se implementa la estructura base del servidor, rutas públicas, servicio de archivos estáticos y persistencia simple mediante archivos planos.

## ¿Por qué elegí `index.js`?
Elegí `index.js` como archivo principal porque es una convención muy utilizada en proyectos Node.js para representar el punto de entrada principal de la aplicación.  
Permite una estructura clara y fácil de identificar para otros desarrolladores.

## Tecnologías usadas
- Node.js
- Express.js
- dotenv
- nodemon

## Requisitos del sistema
- Node.js v18 o superior
- npm
